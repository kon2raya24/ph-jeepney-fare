# ph-jeepney-fare

[![npm version](https://img.shields.io/npm/v/ph-jeepney-fare.svg)](https://www.npmjs.com/package/ph-jeepney-fare)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Philippine jeepney fare calculation utilities

## Installation

```bash
npm install ph-jeepney-fare
```

## Functions

- `computeFare` — Compute jeepney fare based on distance
- `computeDiscountedFare` — Compute discounted fare (student/senior/PWD)
